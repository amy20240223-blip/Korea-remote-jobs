## We are hiring｜网络运维主管（Remote）

我们是一支技术驱动的远程团队，正在招聘一位 **网络运维主管 / Network Operations Lead**，
负责公司整体网络架构、安全体系及团队管理工作。

### 工作方式
- 🌍 远程工作（Remote）
- 🕒 全职
- 👥 管理 10+ 人规模相关团队（网络 / 安全 / 基础设施协作）

---

## 岗位职责

- 规划与设计公司整体网络与安全基础设施，确保网络的 **安全性、稳定性与可扩展性**
- 负责网络架构设计与持续优化，包括网络安全设备的部署、维护及高可用方案
- 主导公司网络 **主 / 被动安全体系** 的整体设计，制定安全策略与技术方案
- 负责大规模网络环境的规划、设计与实施，支持业务持续增长
- 优化网络负载分布与流量调度，降低延迟、避免拥塞
- 管理和指导网络团队的日常工作，制定计划、分配任务并推动落地
- 培养团队成员技术能力，推动技术沉淀、自动化和最佳实践

---

## 任职要求

- 计算机科学、网络安全、信息技术或相关专业，大专及以上学历
- **10 年以上** 网络架构设计与网络安全相关经验
- 具备 **大规模网络环境** 的设计、部署与优化经验
- 在 **负载均衡、网络安全防护** 方面有成熟的实战经验
- 熟悉常见网络协议：TCP/IP、DNS、HTTP 等
- 掌握路由与交换技术，理解虚拟化及云计算网络架构
- 熟练使用网络安全技术与方案：
  - 防火墙 / IDS / IPS
  - VPN / 加密 / 身份认证
- 有第三方网络安全产品与方案的评估和落地经验
- 具备成熟的 **团队管理能力、跨部门协作能力**
- 具备较强的问题分析、应急响应和安全事件处置能力
